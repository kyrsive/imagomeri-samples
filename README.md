# imagomeri-samples

a collection of drum sounds, loops, and oneshots to be used in strudel from the members1 sample pack from [imago meri](https://www.imagomeri.com/) (consisting of members [8strings](https://www.instagram.com/eightstringss/?next=), [anem0s](https://www.instagram.com/anem0s1/?next=), [fluoride](https://www.instagram.com/fluoride_333/?next=), [iora](https://www.instagram.com/iora._x/?next=), [margo proxy](https://soundcloud.com/margoproxy), [nic](https://www.instagram.com/nictorrin/?next=), [nonsixe](https://www.instagram.com/nonsixe/?next=), [phantafiy](https://www.instagram.com/phantafiy/?next=), [quordiav](https://www.instagram.com/quordiav/?next=), [reci.p](https://www.instagram.com/jerrygottherecipe/), [renoncemtn](https://www.instagram.com/renoncemtn/?next=), [rosewither](https://www.instagram.com/rosew1ther/?next=), [shadowpartner](https://www.instagram.com/shadowpartner/?next=), [teek](https://www.instagram.com/10kteek/?next=), [tobi](https://www.instagram.com/_t0biii__/?next=), [wwildergarden](https://www.instagram.com/wwildergarden/?next=), [xfchiba](https://www.instagram.com/xfchiba/?next=), [XOxo](https://www.instagram.com/mytwothumbs/?next=))

to use this inside of strudel, enter the following line in your strudel pattern:
```
samples('github:kyrsive/imagomeri-samples')
```

please support all the sound designers linked 

it's important to note the key of phrases and loops were not found manually so do not trust me on them, you might need to tune them by ear still.
